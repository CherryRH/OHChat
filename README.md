## 项目结构

```
├─ets                                   //代码目录
│  ├─classes                                //所使用对象的类，包含聊天、用户等数据
│  ├─components                             //可复用的ArkTs组件
│  ├─entryability                           //stage模型的activity，可继续添加
│  ├─pages                                  //主要页面，应用入口为index
│  ├─services                               //提供后端服务能力
│  └─utils                                  //数据相关服务
└─resources                             //资源目录
    ├─base
    │  ├─element                            //储存color和string资源
    │  ├─media                              //储存图标、图片等资源
    │  └─profile
    ├─en_US
    │  └─element
    ├─rawfile
    └─zh_CN
        └─element
```
